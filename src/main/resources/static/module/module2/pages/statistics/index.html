批量爬虫


文件格式要求：
1. xls格式的excel文件，文件名统一命名为：“domains.xls”。
2. 表格包含两列，第一行分别为：“学科”、“课程名”。
3. 第二行开始为具体的学科和课程信息，如：“计算机科学”，“数据结构”。
4. 目前只能在idea中运行tomcat可以运行，发布到tomcat服务器上无法正常运行。

单个爬虫

IE浏览器配置：
1. 设置IE浏览器的缩放为100%。
2. “Internet选项”->“安全”：四个安全设置都启用保护模式。
3. 设置IEDriverServer.exe到Config.IE_PATH路径下，默认是放在D盘下。
